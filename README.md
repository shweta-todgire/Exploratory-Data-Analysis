# Airbnb NYC EDA

##  Overview
This project performs Exploratory Data Analysis (EDA) on the Airbnb NYC dataset to understand pricing trends, room types, neighbourhoods, reviews, and listing availability.

## 📂 Dataset
- **Rows:** 48,895
- **Columns:** 16
- **Location:** New York City, USA

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Analysis
- Data Cleaning
- Missing Value Handling
- Descriptive Statistics
- Outlier Detection
- Histogram
- Box Plot
- Scatter Plot
- Bar Chart
- Correlation Heatmap

## 🔍 Key Insights
- Most listings cost **$50–$200** per night.
- Manhattan has the highest average Airbnb price.
- Entire homes/apartments are the most expensive room type.
- Affordable listings receive more reviews.
- 2,972 price outliers were identified.

## ✅ Conclusion
The analysis shows that **location and room type** have a significant impact on Airbnb prices. These insights can help hosts improve pricing strategies and assist travelers in finding accommodations that match their budget.
